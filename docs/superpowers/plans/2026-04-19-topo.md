# Topo Lab Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a new `/lab/topo` page in this Next.js site that ports Rune Skovbo Johansen's "Fast and Gorgeous Erosion Filter" to Three.js with a toggle between 3D terrain and 2D heightmap views, all driven by live Leva parameters.

**Architecture:** One ported GLSL chunk (`erosionShader.ts`) exposes a shared `erodedTerrain(uv) → (height, gradient)` function. A 3D `ShaderMaterial` uses it in the vertex shader to displace a subdivided plane and in the fragment shader to shade using analytical normals. A 2D `ShaderMaterial` uses it in the fragment shader to output a grayscale heightmap. Both materials share a single uniforms object driven by Leva. No render-to-texture; no tests (matches the rest of `src/app/lab/`).

**Tech Stack:** Next.js 15 (static export), React 19, TypeScript, MDX, Three.js r165+, `@react-three/fiber`, `@react-three/drei` (`shaderMaterial`, `OrbitControls`, `OrthographicCamera`), `leva`, existing `ThreeCanvas` + `LevaPanel` + `LabMenu` components. Package manager: `bun`.

**Spec:** `docs/superpowers/specs/2026-04-19-topo-design.md`

## Key project conventions (read before starting)

- Use `bun` (not `npm`). Install: `bun install`. Dev server: `bun dev`.
- Lab pages live at `src/app/lab/<slug>/`. Route auto-resolves to `/lab/<slug>`.

Files to create (all under `src/app/lab/topo/`):

- `page.mdx` — MDX page with title, intro, `<Canvas />`.
- `TopoCanvas.tsx` — default-exported client canvas component. Owns `view: "2d" | "3d"` state, the Leva controls hook, and the shared uniforms object. Mounts `<ThreeCanvas>` with the selected subview.

- `terrainMaterial.ts` — `TerrainMaterial = shaderMaterial(...)` for the 3D view. Includes `erosionShaderChunk` in both vertex and fragment. Vertex displaces; fragment shades from analytical normals.
- `heightmapMaterial.ts` — `HeightmapMaterial = shaderMaterial(...)` for the 2D view. Fragment calls `erodedTerrain` and outputs grayscale.
- `TerrainMesh.tsx` — R3F component mounting the subdivided plane + `<terrainMaterial />` + `<OrbitControls />`.
- `HeightmapQuad.tsx` — R3F component mounting a fullscreen quad + `<heightmapMaterial />` + `<OrthographicCamera makeDefault />`.
- `ViewToggle.tsx` — small styled button that toggles the view state. Positioned absolutely over the canvas container.
- `TopoCanvas.module.css` — styles for the view-toggle button and any wrapper tweaks.

No existing files are modified.

```ts
// Placeholder during scaffolding. Replaced in Task 6 by the ported Shadertoy
// "Advanced Terrain Erosion Filter" source (view wXcfWn), exposing the same
// erodedTerrain(vec2 uv) entry point but with the real algorithm + uniforms.
export const erosionShaderChunk = /* glsl */ `
  uniform float uBaseAmplitude;
  uniform float uBaseFrequency;
  uniform int uBaseOctaves;

  // 2D hash
  float hash21(vec2 p) {
    p = fract(p * vec2(123.34, 456.21));
    p += dot(p, p + 45.32);
    return fract(p.x * p.y);
  }

  // Value noise
  float valueNoise(vec2 p) {
    vec2 i = floor(p);
    vec2 f = fract(p);
    vec2 u = f * f * (3.0 - 2.0 * f);
    float a = hash21(i);
    float b = hash21(i + vec2(1.0, 0.0));
    float c = hash21(i + vec2(0.0, 1.0));
    float d = hash21(i + vec2(1.0, 1.0));
    return mix(mix(a, b, u.x), mix(c, d, u.x), u.y);
  }

  // Placeholder fBM-style height. Returns (height, dH/dx, dH/dy) where
  // gradients are finite-differenced — good enough to validate the pipeline.
  vec3 erodedTerrain(vec2 uv) {
    float freq = uBaseFrequency;
    float amp = uBaseAmplitude;
    float h = 0.0;
    for (int i = 0; i < 8; i++) {
      if (i >= uBaseOctaves) break;
      h += amp * valueNoise(uv * freq);
      freq *= 2.0;
      amp *= 0.5;
    }
    float eps = 1.0 / 512.0;
    float hx = 0.0;
    float hy = 0.0;
    {
      float f = uBaseFrequency; float a = uBaseAmplitude;
      for (int i = 0; i < 8; i++) {
        if (i >= uBaseOctaves) break;
        hx += a * valueNoise((uv + vec2(eps, 0.0)) * f);
        hy += a * valueNoise((uv + vec2(0.0, eps)) * f);
        f *= 2.0; a *= 0.5;
      }
    }
    return vec3(h, (hx - h) / eps, (hy - h) / eps);
  }
`;
```

Write `src/app/lab/topo/terrainMaterial.ts`:

```ts
import { shaderMaterial } from "@react-three/drei";

import { erosionShaderChunk } from "./erosionShader";

export const TerrainMaterial = shaderMaterial(
  {
    uBaseAmplitude: 0.3,
    uBaseFrequency: 2.0,
    uBaseOctaves: 5,
    uDisplacementScale: 1.0
  },
  /* glsl */ `
    varying vec3 vNormalW;
    varying float vHeight;

    ${erosionShaderChunk}

    void main() {
      vec3 ht = erodedTerrain(uv);
      float h = ht.x;
      vec3 displaced = position + vec3(0.0, 0.0, h * uDisplacementScale);

      // Analytical normal from gradient of height in UV space.
      // Plane is 2x2 in XY, uv in [0,1], so d(position.xy)/d(uv) = (2, 2).
      // dHeight/dx_world = (ht.y / 2) * uDisplacementScale; similarly for y.
      float s = uDisplacementScale * 0.5;
      vec3 n = normalize(vec3(-ht.y * s, -ht.z * s, 1.0));

      vNormalW = normalize(mat3(modelMatrix) * n);
      vHeight = h;
      gl_Position = projectionMatrix * modelViewMatrix * vec4(displaced, 1.0);
    }
  `,
  /* glsl */ `
    varying vec3 vNormalW;
    varying float vHeight;

    void main() {
      vec3 lightDir = normalize(vec3(0.4, 0.6, 0.7));
      float lambert = clamp(dot(normalize(vNormalW), lightDir), 0.0, 1.0);
      float ambient = 0.25;
      vec3 base = mix(vec3(0.78, 0.78, 0.78), vec3(0.95, 0.95, 0.95), clamp(vHeight * 1.5 + 0.5, 0.0, 1.0));
      vec3 color = base * (ambient + (1.0 - ambient) * lambert);
      gl_FragColor = vec4(color, 1.0);
    }
  `
);
```

import { OrbitControls } from "@react-three/drei";
import { extend, useFrame } from "@react-three/fiber";
import { useMemo, useRef } from "react";
import * as THREE from "three";

import { TerrainMaterial } from "./terrainMaterial";
import type { TopoUniforms } from "./uniforms";

extend({ TerrainMaterial });

type Props = {
  uniforms: TopoUniforms;
};

export function TerrainMesh({ uniforms }: Props) {
  const geometry = useMemo(
    () => new THREE.PlaneGeometry(2, 2, 256, 256),
    []
  );
  const materialRef = useRef<THREE.ShaderMaterial>(null);

  useFrame(() => {
    const m = materialRef.current;
    if (!m) return;
    m.uniforms.uBaseAmplitude.value = uniforms.uBaseAmplitude.value;
    m.uniforms.uBaseFrequency.value = uniforms.uBaseFrequency.value;
    m.uniforms.uBaseOctaves.value = uniforms.uBaseOctaves.value;

- [ ] **Step 2: Port the GLSL into `erosionShaderChunk`**

Transform the pasted source as follows, writing the result into `src/app/lab/topo/erosionShader.ts` (replacing the placeholder):

1. Concatenate the Common tab contents first, then any Buffer tabs whose logic is needed for the final Image (most of this filter is single-pass; skip Buffer tabs that are only used for side displays like the ridge map we declared out-of-scope).
2. Remove any `#version` directive, `precision` directives, and Shadertoy-specific built-ins that Three.js will already provide.
3. Replace hardcoded parameter constants (look for `const float`, `#define`, or magic numbers inside named sections like "Erosion Params") with `uniform` declarations. Each becomes a line at the top of the chunk: `uniform float uErosionStrength;` etc.
4. Replace the Image tab's `mainImage(out vec4 fragColor, in vec2 fragCoord)` with a function `vec3 erodedTerrain(vec2 uv)` whose body:
   - Uses `uv` directly as its sample position (do NOT divide by `iResolution` — that conversion happened at the caller).
   - Drops any display-only logic (grayscale conversion, color ramps, UI overlays).
   - Returns `vec3(height, dHeight_du, dHeight_dv)`. If the Shadertoy already computes analytical derivatives (it does — the post emphasizes this), return them; otherwise finite-difference as the placeholder does.
5. Rename any identifiers that collide with common Three.js built-ins (e.g. `normal`, `position`) inside the chunk to avoid shader compile issues.
6. Remove any `iTime`/`iResolution`/`iMouse` references — none of the filter's logic should depend on time; if it does, replace with a new `uniform float uTime` that we drive from `useFrame`.

After this edit, `erosionShaderChunk` is a self-contained GLSL string starting with all needed `uniform` declarations and ending with `vec3 erodedTerrain(vec2 uv) { ... }`.
